## MENU STRUCTURE
### 1. PLANS
Opens "Plans & Dessins" file browser
- Source: Project media section + uploaded plan files
- API: GET /api/projects/:projectId
- Field: `plansDrawingsLink` (external link if set)
- Files: Filter archived files by "plans" category
### 2. DQE
Scrollable DQE item list with filters
- Source: project.items array
- API: GET /api/projects/:projectId
**Filter Options:**
- By Lot: Use `lotCode` field
- By Contractor: Use assigned contractor data
```typescript
// Filter by lot
const filteredByLot = project.items.filter(item => item.lotCode === selectedLot);
// Filter by contractor (if assigned)
const filteredByContractor = project.items.filter(item => item.contractorId === selectedContractorId);
